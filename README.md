# მიხო 🥟 — ხინკლის სათავგადასავლო (Super Mikho)

მარიოს სტილის პლატფორმერი ქართული თემატიკით.

## პერსონაჟები და ელემენტები
- **მიხო** — მთავარი გმირი, მამაცი კახელი: ულვაშები, კახური ქუდი და ჯვარი კისერზე.
- **ხინკალი** 🥟 — შესაგროვებელი (სოკოს ნაცვლად).
- **ჯუჯა ყიზილბაშები** — მტრები, რომელთაც გადახტომით (თავზე დახტომით) ამარცხებ.
- **ჯვარი** 🏁 — ტურის ფინიში.

## მართვა
- `← →` ან `A / D` — სიარული
- `Space` / `↑` / `W` — ხტომა
- მტერს თავზე დახტომით ამარცხებ; გვერდიდან შეხება სიცოცხლეს გართმევს.

## გაშვება
უბრალოდ გახსენი `index.html` ბრაუზერში. ან:

```bash
python3 -m http.server 8000
# შემდეგ გახსენი http://localhost:8000
```

## სტატუსი
დასაწყისისთვის გაკეთებულია **პირველი ტური**. შემდეგ ტურებს ვამატებთ `game.js`-ში `LEVEL_*` რუკების დამატებით.
